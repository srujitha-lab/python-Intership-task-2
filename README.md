## API INTEGRATION PROJECT(WEATHER/CRYPTO/NEWS)
## Project Overview
This project demonstrates how to fetch real time data from APIs using python.It integrates weather,crypto currency and news APIs , processes,JSON responses, and displays useful information in a user friendly format.
## Goal 
-fetch API data using python
-parse JSON responses
-apply search/filter functionality
-display formatted output
## Technologies used
Python 3.x
request module
JSON handling
## APIs used
-Weather API 
-Crypto API
-News API
## How to run
Run the python script: python main.py
## Features
-Fetch real time API data
-Search/filter news by keyword
-Displays structured output
## Common Issues
-Invalid API key
-No data found
-No columns to parse
